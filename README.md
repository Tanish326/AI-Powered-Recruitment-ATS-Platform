# AI-Powered Recruitment ATS Platform

An AI-powered Applicant Tracking System (ATS) designed to simplify the recruitment process. The platform enables recruiters to create job postings, manage candidates, and leverage AI-assisted resume analysis to make hiring more efficient.

---

## 🚀 Features

* Secure User Authentication
* Job Posting and Management
* Candidate Application Tracking
* Recruiter Dashboard
* AI-Powered Resume Analysis
* Responsive User Interface
* RESTful API Integration

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Tanish326/AI-Powered-Recruitment-ATS-Platform.git
cd AI-Powered-Recruitment-ATS-Platform
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```
